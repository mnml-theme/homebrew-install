_[Homebrew]_ `install` [mnml]
=============================
Homebrew [formulae] for **m**i**n**i**m**a**l** themes.

Install
-------
~~~ sh
brew tap mnml-theme/install
brew install svn mnml-theme
~~~
or `brew bundle` with _[Brewfile]_:
~~~ rb
tap "mnml-theme/install"
brew "svn"
brew "mnml-theme"
~~~
