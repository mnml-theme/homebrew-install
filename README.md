_[Home`brew`]_ `install` [mnml]
===============================
Home`brew` [formulae] for **m**i**n**i**m**a**l** themes.

Install
-------
~~~ sh
brew tap mnml-theme/install
brew install svn mnml-theme
~~~
or `brew bundle install`[` --file`] from _[Brewfile]_:
~~~ rb
tap  "mnml-theme/install"
brew "svn"
brew "mnml-theme"
~~~
